credSuccessHandler

()

Defined in src\js\src\bootstrap.js:108

Success handler for the Credential request. Method checks the backend session access tracker. If the session tracker is non-zero, this method updates the application state with the value from the session tracker. This method also checks the application state for the last session access timestamp. If a timestamp is present and is within session length, this method assumes a valid session and skips authentication. Otherwise, the authentication process is started.
